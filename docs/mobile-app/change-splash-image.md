# How to change splash screen image

Go to `assets/svg/Logo` folder. You can change your image to exist image with same name.

:::note
We recommend using .svg format, but you can also use .png, .jpg, or .jpeg if you adjust the configuration in `lib/ui/screens/splash_screen.dart`.
:::
